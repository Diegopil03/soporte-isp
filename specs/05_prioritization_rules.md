# Reglas de priorización

## Prioridades
- Alta
- Media
- Baja

## Reglas
1. Afectación múltiple tiene prioridad máxima
2. Residente > casa de descanso
3. Corte total > parcial
4. Prioridad definida
5. Antigüedad del ticket
6. Bloque de tiempo

## Ejemplo
Dos cortes: se atiende primero el residente aunque haya llegado después
